# Final-Project

# Features:
* Movement (walking, sprinting, jumping, crouching)
* Interaction system
* Picking up (and throwing) objects
* Different RigidBody physics
* Shaders and lighting effects

# Instructions:
* Movement: WASD
* Jump: SPACE
* Walk: ALT
* Sprint: CTRL
* Interact: E
* Pick up/leave: F
* Throw: LMB (hold)
* Reset: R
